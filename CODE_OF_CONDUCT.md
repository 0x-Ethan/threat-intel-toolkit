# Code of Conduct

## Standards

This project is maintained as a professional security research resource. Contributors and participants are expected to:

- Engage with technical accuracy and intellectual honesty
- Respect the legal and ethical boundaries defined in [DISCLAIMER.md](DISCLAIMER.md)
- Provide constructive, evidence-based feedback
- Acknowledge the work of others appropriately

## Unacceptable Behavior

The following are not tolerated:

- Harassment, personal attacks, or discriminatory language
- Sharing or encouraging unauthorized access to systems
- Misrepresenting research findings or credentials
- Using this project's community for commercial solicitation

## Scope

This Code of Conduct applies within GitHub Issues, Pull Requests, Discussions, and any other project-affiliated communication channels.

## Enforcement

Violations may result in removal of contributions and blocking from the project. For serious concerns, contact `[EMAIL]`.
