# Research Notes

This directory contains published research, methodology documentation, and responsible disclosure records produced in connection with this toolkit and broader threat intelligence work.

## Structure

```
research-notes/
├── methodology/        # Technical methodology and framework notes
├── disclosures/        # Post-remediation vulnerability disclosure write-ups
└── frameworks/         # Privacy and compliance framework analysis notes
```

## Publication Standard

Research published here meets the following criteria:

- **Passive or authorized scope only** — no unauthorized access to any system
- **Post-remediation disclosures** — vulnerabilities are only published after vendor has deployed a fix or the disclosure period has elapsed
- **Framework-referenced** — findings are mapped to MITRE ATT&CK, NIST CSF, and relevant privacy frameworks where applicable
- **Reproducible** — methodology is documented clearly enough for peer review
